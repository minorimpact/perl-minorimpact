# NAME

MinorImpact::reference

# DESCRIPTION

An object for other linking two MinorImpact objects.

## toString

Overrides the `list` format to use `reference_list` instead of
`object_list`.

See [MinorImpact::Object::toString()](./MinorImpact_Object.md#tostring).
